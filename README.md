# 📡 Email Radar

**Email Radar** is an AI-powered web app that helps you draft, analyze, and improve emails with clarity, tone, and professionalism — all in seconds. Whether you're writing a cold outreach, a follow-up, or a thank-you note, Email Radar gives you the edge.

---

## 🚀 Features

- ✍️ **AI Email Generator**  
  Get email drafts instantly based on your intent, audience, and tone.

- 🧠 **Smart Feedback**  
  Analyze any email for tone, clarity, grammar, and structure.

- 🗂️ **Save & Reuse Drafts**  
  Store personalized drafts and reuse them when needed.

- 🎨 **Custom Tones**  
  Choose between tones: professional, friendly, assertive, empathetic, etc.

- 🔐 **Google Sign-In**  
  Secure authentication using Firebase Auth.
